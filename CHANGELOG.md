# 更新日志

所有显著的变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，版本遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [Unreleased]

### 新增

- **Windows 系统支持**：添加 PowerShell 脚本 `install.ps1`，兼容 Windows 7/10/11。
  - DNS 优化：通过 `Set-DnsClientServerAddress` 修改网络适配器 DNS。
  - 开发工具镜像优化：为 npm、pip、Docker 配置国内镜像。
  - GitHub 访问优化：更新 hosts 或设置代理环境变量。
  - 回滚功能：备份注册表相关项和配置文件，支持按备份清单恢复。
  - 交互式菜单和命令行参数（与 Linux 版本保持一致）。
- **Windows 版交互式语言选择**：在未指定 `-lang` 参数时自动询问用户选择语言。
- **全球镜像源数据库**：按大洲/国家分类，覆盖亚洲（中国及东亚/东南亚）、欧洲、北美洲、南美洲、大洋洲、非洲，总计超过100个镜像站。
- **地区选择菜单**：在系统镜像优化模块中，用户可先选择地区，再选择具体镜像源，提供更便捷的浏览体验。
- **国家分组显示**：在地区内按国家分组展示镜像，方便用户快速定位。
- **协议选择**：当镜像源同时支持http/https时，允许用户选择协议。
- **命令行快捷指定镜像**：支持 `--mirror` 参数直接使用镜像域名或预定义名称（如 `aliyun`）。

### 修复

- **Windows 版脚本语法错误**：修正了 PowerShell 中的括号不匹配、字符串分割错误，确保脚本可正常运行。
- **Windows 版语言显示问题**：将 DNS 优化菜单中的服务商名称（如 Aliyun → 阿里云）、提示信息全面中文化。
- **语言变量作用域**：修复 `Ask-Language` 函数中未正确修改全局语言变量的问题，确保语言选择后菜单能立即切换。
- **Linux 版数组引用错误**：修复 `select_mirror_from_region` 函数中间接引用数组只取第一个元素的问题，使用 `eval` 安全展开数组。
- **apply_mirror 参数调用**：修正了从 `run_system_mirror` 调用 `apply_mirror` 时的参数顺序，确保镜像名称正确传递。
- **远程执行时函数定义顺序**：调整主脚本结构，确保所有函数在 `main` 调用前定义，避免 `command not found` 错误。
- **颜色变量未定义问题**：在 `system_mirror.sh` 中添加默认颜色定义，以防 `common.sh` 未加载颜色。

### 变更

- **镜像源数据结构**：统一使用 `地区|国家|名称|域名|协议|备注` 格式，便于扩展和维护。
- **地区选择逻辑**：由单一源列表改为分层选择，提升用户体验。
- **交互流程优化**：在未指定语言且为交互模式时，自动询问用户选择语言（Linux/Windows 均已支持）。
- **Windows 版输出函数**：新增 `Write-Option` 和增强 `Write-I18n`，使菜单和提示信息更易于维护。
